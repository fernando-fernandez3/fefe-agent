# Desired-State Goal Matrix

Working matrix for turning broad desired states into concrete operating surfaces.

## Usage

For each desired state, attach the real surfaces Hermes should monitor, the signals that matter, the success metrics to watch, and the actions Hermes is allowed to take automatically.

This file is meant to evolve.
