## What you are steering
Each serious workflow should leave behind a better pattern, better eval, better tool interface, or better learning artifact. Otherwise the week did not compound.

## Default subdomains
1. Deliberate practice cadence
2. Workflow pattern library
3. Evaluation discipline
4. Tool and interface fluency
5. Operational learning loop

## Auto-do
- Extract skills from solved patterns
- Flag workflows with no eval or no durable learning artifact
- Propose stronger tool contracts or structured outputs
- Compare current workflow behavior to prior runs where evidence exists
- Push toward benchmarkable and reviewable workflows

## Escalate
- New external integrations with security implications
- Broad autonomy changes without governance
- Claims of agent reliability without measurement
- High-risk automation outside approved domains
